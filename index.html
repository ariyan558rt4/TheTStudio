دربارهٔ ما

استودیوی فیروزه یه گروه برنامه‌نویسی هست که سال ۱۴۰۴ تأسیس شده و هدفش ساخت برنامه و آموزش هست.

راهنمای کوتاه VS Code

  1. نصب افزونه‌ها: از نوار کناری روی آیکن Extensions بزن و افزونه‌های «Persian RTL Support» و «Prettier» رو نصب کن.
  2. ساخت پروژهٔ سریع: کلیدهای Ctrl+Shift+P رو بزن، بنویس New File و یک فایل index.html بساز. تایپ کن ! و Enter تا قالب HTML ساخته بشه.
  3. اجرای زنده: افزونهٔ «Live Server» رو نصب کن و روی فایل راست‌کلیک > Open with Live Server تا سایتت محلی اجرا بشه.
  4. فرمت خودکار: از Status Bar آیکن Prettier رو فعال کن تا کدت مرتب بشه. کلید میانبر: Shift+Alt+F.
  5. میانبرها: ترمینال یکپارچه: Ctrl+`، جستجو در پروژه: Ctrl+Shift+F.

  <button class="eq" onclick="calcEval()">=</button>
</div>
<script>
  const display = document.getElementById('calc-display');
  function calcInput(v){ display.value += v }
  function calcClear(){ display.value = '' }
  function calcEval(){ try{ display.value = Function('return '+display.value)() } catch{ alert('ورودی نامعتبر!') } }
</script>
